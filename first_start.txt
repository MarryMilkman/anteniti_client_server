Первое включение мастера:
1)Включаем питание
	-соднается скрытая WiFi сеть с SSID = "XXX" и паролем = "мак-адрес роутера"
2)Сканируем QR-код
	-из QR-кода получаем данные о роутере
	-открывается экран настройки
3)Настраиваем роутер
4)Нажимаем кнопку применить
	-по введённым настройкам преложение по шаблону "mester" создаёт файл конфигураций
	-из приложения происходит подключение к скрытому WiFi "XXX"
	-локально подключаемся к роутеру, через порт "Y", на котором будет висеть "сервер локальной настройки"
	-забрасываются файл конфигураций, апк клиент-сервера, скрипты
	-роутер применяет конфигурации, запускает клиент-сервер
	-включается основной WiFi, на котором уже должен быть интернет
	?если интернета не будет -> роутер не сможет стучатся на Cloud -> нельзя будет изменить настройки


Первое включение сателита:
1)Включаем питание
	-соднается скрытая WiFi сеть с SSID = "XXX" и паролем = "мак-адрес роутера"
2)В приложении выбираем пункт "добавить новое устройство в меш"
3)Сканируем QR-код
	-из QR-кода получаем данные о роутере
	-открывается экран настройки
4)Настройка роутера
5)Нажимаем кнопку применить
	-по введённым настройкам преложение по шаблону "slave" создаёт файл конфигураций
	-из приложения происходит подключение к скрытому WiFi "XXX"
	-локально подключаемся к роутеру, через порт "Y", на котором будет висеть "сервер локальной настройки"
	-забрасываются файл конфигураций, апк клиент-сервера, скрипты
	-роутер применяет конфигурации, связывается в меш сеть, запускает клиент-сервер
	-включается основной WiFi, на котором уже должен быть интернет


**************
Сервер локальной настройки - сервер, который, при отсутствии интернета, слушает "Y" порт на локальные соединения.
	Из приложения в него можно залить файл конфигураций, клиент сервер и тд.
